# Berechnungsbeispiele Forschungszulage

## Inhaltsverzeichnis

1. Eigenbetriebliche Forschung - Kleines Unternehmen
2. Eigenbetriebliche Forschung - Mittelstand (KMU)
3. Auftragsforschung
4. Kombiniert: Eigenbetrieblich + Auftrag
5. Einzelunternehmer mit Eigenleistung
6. Maximale Foerderung ausschoepfen

---

## Beispiel 1: Kleines Software-Unternehmen (KMU, ab 2026)

**Szenario**: Ein Startup mit 15 Mitarbeitern entwickelt einen neuartigen NLP-Algorithmus. Zwei Entwickler arbeiten 12 Monate Vollzeit am FuE-Projekt.

| Position                       | Berechnung     | Betrag          |
| ------------------------------ | -------------- | --------------- |
| Bruttolohn Entwickler 1        | 12 x 6.500 EUR | 78.000 EUR      |
| Bruttolohn Entwickler 2        | 12 x 5.800 EUR | 69.600 EUR      |
| **Lohnkosten gesamt**          |                | **147.600 EUR** |
| Gemeinkostenpauschale 20%      | 147.600 x 0,20 | 29.520 EUR      |
| **Bemessungsgrundlage**        |                | **177.120 EUR** |
| **Forschungszulage (35% KMU)** | 177.120 x 0,35 | **61.992 EUR**  |

---

## Beispiel 2: Mittelstaendischer Maschinenbauer (KMU, ab 2026)

**Szenario**: 180 Mitarbeiter, ein Team von 8 Ingenieuren arbeitet an einem neuen Fertigungsverfahren. Durchschnittlich 60% ihrer Arbeitszeit entfaellt auf FuE.

| Position                       | Berechnung                  | Betrag          |
| ------------------------------ | --------------------------- | --------------- |
| Bruttolohnkosten 8 Ingenieure  | 8 x 75.000 EUR Jahresgehalt | 600.000 EUR     |
| FuE-Anteil 60%                 | 600.000 x 0,60              | 360.000 EUR     |
| Gemeinkostenpauschale 20%      | 360.000 x 0,20              | 72.000 EUR      |
| **Bemessungsgrundlage**        |                             | **432.000 EUR** |
| **Forschungszulage (35% KMU)** | 432.000 x 0,35              | **151.200 EUR** |

---

## Beispiel 3: Auftragsforschung

**Szenario**: Ein Unternehmen beauftragt eine Universitaet mit einem Forschungsprojekt. Auftragsvolumen: 200.000 EUR.

| Position                     | Berechnung     | Betrag          |
| ---------------------------- | -------------- | --------------- |
| Auftragskosten               |                | 200.000 EUR     |
| Foerderfaehiger Anteil (70%) | 200.000 x 0,70 | 140.000 EUR     |
| Gemeinkostenpauschale 20%    | 140.000 x 0,20 | 28.000 EUR      |
| **Bemessungsgrundlage**      |                | **168.000 EUR** |
| **Forschungszulage (25%)**   | 168.000 x 0,25 | **42.000 EUR**  |

Hinweis: Der Auftraggeber stellt den Antrag und erhaelt die Zulage.

## Beispiel 4: Kombiniert (eigenbetrieblich + Auftragsforschung, KMU)

**Szenario**: Ein KMU forscht intern und beauftragt zusaetzlich ein Fraunhofer-Institut.

| Position                        | Berechnung                 | Betrag          |
| ------------------------------- | -------------------------- | --------------- |
| Eigene Lohnkosten FuE           | 3 Mitarbeiter x 65.000 EUR | 195.000 EUR     |
| Auftragskosten Fraunhofer       |                            | 150.000 EUR     |
| Foerderfaehig Auftrag (70%)     | 150.000 x 0,70             | 105.000 EUR     |
| **Foerderfaehige Direktkosten** |                            | **300.000 EUR** |
| Gemeinkostenpauschale 20%       | 300.000 x 0,20             | 60.000 EUR      |
| **Bemessungsgrundlage**         |                            | **360.000 EUR** |
| **Forschungszulage (35% KMU)**  | 360.000 x 0,35             | **126.000 EUR** |

## Beispiel 5: Einzelunternehmer mit Eigenleistung (ab 2026)

**Szenario**: Ein freiberuflicher Ingenieur forscht selbst. Er arbeitet 20 Stunden pro Woche an seinem FuE-Projekt ueber 10 Monate.

| Position                       | Berechnung                | Betrag          |
| ------------------------------ | ------------------------- | --------------- |
| Eigenleistung                  | 20h x 43 Wochen x 100 EUR | 86.000 EUR      |
| Gemeinkostenpauschale 20%      | 86.000 x 0,20             | 17.200 EUR      |
| **Bemessungsgrundlage**        |                           | **103.200 EUR** |
| **Forschungszulage (35% KMU)** | 103.200 x 0,35            | **36.120 EUR**  |

Hinweis: Max. 40 Stunden pro Woche ansetzbar. Der Stundensatz von 100 EUR gilt ab 2026 (vorher 70 EUR).

## Beispiel 6: Maximale Foerderung (Grossunternehmen, ab 2026)

**Szenario**: Ein Grossunternehmen mit umfangreicher FuE-Abteilung.

| Position                            | Berechnung        | Betrag             |
| ----------------------------------- | ----------------- | ------------------ |
| FuE-Lohnkosten                      |                   | 8.000.000 EUR      |
| Auftragsforschung (70%)             |                   | 3.000.000 EUR      |
| Foerderfaehige Direktkosten         |                   | 11.000.000 EUR     |
| Gemeinkostenpauschale 20%           |                   | 2.200.000 EUR      |
| Summe                               |                   | 13.200.000 EUR     |
| **Bemessungsgrundlage (gedeckelt)** |                   | **12.000.000 EUR** |
| **Forschungszulage (25%)**          | 12.000.000 x 0,25 | **3.000.000 EUR**  |

Bei einem KMU waere die maximale Zulage: 12.000.000 x 0,35 = **4.200.000 EUR**.

## Beispiel 7: Kleine GmbH mit zwei Geschaeftsfuehrern (KMU, ab 2026)

**Szenario**: Zwei GF-Gesellschafter einer Software-GmbH arbeiten ca. 70% ihrer Zeit an FuE-Projekten. Zusaetzlich wird ein externer Freelancer eingesetzt.

| Position                               | Berechnung           | Betrag          |
| -------------------------------------- | -------------------- | --------------- |
| GF-Gehalt 1 (Brutto inkl. AG-Anteil)   | 80.000 EUR x 70% FuE | 56.000 EUR      |
| GF-Gehalt 2 (Brutto inkl. AG-Anteil)   | 80.000 EUR x 70% FuE | 56.000 EUR      |
| **Eigene Lohnkosten FuE**              |                      | **112.000 EUR** |
| Externer Freelancer (Rechnungen netto) | 40.000 EUR           | 40.000 EUR      |
| Foerderfaehig Auftrag (70%)            | 40.000 x 0,70        | 28.000 EUR      |
| **Foerderfaehige Direktkosten**        |                      | **140.000 EUR** |
| Gemeinkostenpauschale 20%              | 140.000 x 0,20       | 28.000 EUR      |
| **Bemessungsgrundlage**                |                      | **168.000 EUR** |
| **Forschungszulage (35% KMU)**         | 168.000 x 0,35       | **58.800 EUR**  |

**Achtung bei GmbH**: Die Forschungszulage von 58.800 EUR ist als Ertrag steuerpflichtig. Bei einem kombinierten Steuersatz von ca. 30% (KSt + GewSt + SolZ) verbleibt eine **Nettofoerderung von ca. 41.160 EUR**.

**Warum nicht 100% FuE-Anteil?** Das Finanzamt erwartet, dass Geschaeftsfuehrer einen Teil ihrer Zeit fuer Verwaltung, Akquise, Buchhaltungskoordination etc. aufwenden. 70% FuE ist bei einer Zwei-Personen-GmbH realistisch und vertretbar. 100% waere schwer durchsetzbar.

---

## Wichtige Hinweise zur Berechnung

1. **Nur tatsaechlicher FuE-Anteil**: Bei Mitarbeitern die nur teilweise in FuE arbeiten, zaehlt nur der FuE-Anteil der Arbeitszeit. Dies muss dokumentiert werden.

2. **Bruttolohn als Basis**: Foerderfaehig sind die dem Lohnsteuerabzug unterliegenden Loehne - also Bruttolohn inklusive AG-Anteil Sozialversicherung.

3. **Bemessungsgrundlage pro Wirtschaftsjahr**: Die 12 Mio. EUR Obergrenze gilt pro Wirtschaftsjahr, nicht pro Projekt.

4. **Verbundene Unternehmen**: Bei verbundenen Unternehmen gilt die Bemessungsgrundlage fuer den gesamten Verbund.

5. **Keine Doppelfoerderung**: Aufwendungen die bereits durch andere oeffentliche Foerdermittel gedeckt sind, duerfen nicht nochmal angesetzt werden.

6. **Gemeinkostenpauschale ab 2026**: Gilt nur fuer Projekte die ab 01.01.2026 beginnen. Fuer aeltere Projekte gilt die Pauschale nicht.

7. **Steuerpflicht bei Kapitalgesellschaften (GmbH/AG)**: Die Forschungszulage ist bei Kapitalgesellschaften als Betriebsvermoegenssmehrung steuerpflichtig. Der Nettoeffekt ist daher ca. 70% des Bruttobetrags (abhaengig vom individuellen Steuersatz). Bei Personengesellschaften und Einzelunternehmen ist die steuerliche Behandlung guenstiger.

8. **Geschaeftsfuehrergehaelter**: Bei GmbH-Geschaeftsfuehrern sind die dem Lohnsteuerabzug unterliegenden Gehaelter foerderfaehig, sofern angemessen. Das Finanzamt erwartet einen plausiblen Verwaltungsanteil. 100% FuE-Ansatz ist auch bei kleinen Unternehmen schwer durchsetzbar.

9. **Externe Dienstleister / Freelancer**: Zaehlen als Auftragsforschung (70% der Nettokosten). Effektiver Foerdersatz: KMU 24,5%, Nicht-KMU 17,5%. Der Auftragnehmer darf nicht weiter unterbeauftragen (Unterauftraege nicht foerderfaehig).

10. **Rueckwirkend moeglich**: Aufwendungen koennen bis zu 4 Jahre rueckwirkend geltend gemacht werden. Beispiel: Im Jahr 2026 koennen noch Aufwendungen aus 2022 beantragt werden.
