# Dutch

Support `nl-NL` and `nl-BE`.

## Quotation marks

Dutch has no universal rule that requires single or double quotation marks.
Editorial styles vary, and many use single curly quotes first:

```text
‘Hij zei: “Ik kom morgen.”’
```

Preserve a consistent existing hierarchy. If none exists, prefer single outer
and double nested quotes for general editorial prose. Place punctuation
according to whether it belongs to the quotation or the surrounding sentence.

## Punctuation and spacing

- Put no space before `, . : ; ? !`.
- Put spaces around a parenthetical dash: `Het resultaat – na twee pogingen – was goed.`
- Use the same dash consistently for ranges and relationships; do not substitute
  a hyphen for a typographic dash in finished prose.
- Use curly apostrophes where spelling requires one. Preserve apostrophes in
  forms such as `’s`, `’t`, and elided names.
- Use a real ellipsis and spaces according to sentence structure; do not run
  blind substitutions through source code.
- Keep a number and its unit or symbol together when the format places them
  next to each other.

## Regional handling

Do not treat Belgian Dutch as French. Apply Dutch punctuation to `nl-BE`, then
use Belgian regional data for dates, numbers, and currency. Preserve Flemish or
Netherlands terminology as written; vocabulary normalization is outside this
skill.
