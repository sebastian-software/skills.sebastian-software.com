What makes the collection different

Plausible is easy. Reliable takes a system.

The skills encode the professional decisions that generic advice usually leaves implicit.

  1. 01

    Practice-built

    Distilled from engineering, design review, product work, consulting, and repeated agent sessions.

  2. 02

    Evidence-led

    Repository state, user data, research, and observed behavior outrank confident assumptions.

  3. 03

    Outcome-oriented

    Discovery, implementation, review, verification, and delivery stay connected.

  4. 04

    Progressively disclosed

    Agents load the smallest useful route and references, not the whole library every time.

  5. 05

    Explicit about boundaries

    Every skill states what it must not invent and when a specialist workflow is needed.

Start with one. Grow deliberately.

Install the skill you need today.

Both the skills CLI and DALO support selective installation. You do not need to adopt the full collection to use one skill.

01

Quick install

skills CLI

Try one skill immediately with any compatible agent environment.

npx skills add sebastian-software/skills.sebastian-software.com --skill effective-web

Replace effective-web with any skill name, or pass --all.

02

Managed setup

DALO

Install DALO, initialize its configuration, link an agent target, and pin the reviewed selection.

curl -fsSL https://dalo.sh/install.sh | sh dalo init dalo target link codex dalo source add-catalog sebastian https://github.com/sebastian-software/skills.sebastian-software.com.git dalo source select sebastian effective-web dalo approve skill sebastian:effective-web dalo sync

Catalog selections stay pinned; new repository skills never activate silently.
