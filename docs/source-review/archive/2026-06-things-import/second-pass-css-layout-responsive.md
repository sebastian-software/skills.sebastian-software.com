## Review Notes

- User preference established: move faster by deciding high-confidence sources
  directly and only pausing for discussion when the classification is genuinely
  uncertain.
- Prefer finer-grained references: `responsive-design` for cross-device and
  content adaptation policy, `css-layout-responsive` for CSS layout primitives
  and production layout recipes, `component-development` for reusable component
  construction, and `baseline` for support-policy decisions.
